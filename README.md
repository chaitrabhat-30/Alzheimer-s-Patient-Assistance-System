# Alzheimer's-Patient-Assistance-System

A web-based assistive technology platform designed to support independence 
for Alzheimer's patients through real-time facial recognition, automated 
daily reminders, and cognitive memory games.

## Features
- 🎯 Real-time facial recognition for caregiver/family identification
- ⏰ Daily reminder scheduling with automated alert triggers
- 🧠 Cognitive memory games to support mental engagement
- 🌐 Accessible, user-friendly web frontend

## Tech Stack
- **Backend:** Python (Flask)
- **Computer Vision:** OpenCV
- **Database:** SQLite
- **Frontend:** HTML, CSS

## Setup
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Add your own API keys in `appp.py` (see placeholders)
4. Run: `python appp.py`

## 🔒 Security Note
For safety, the Telegram Bot Token and Google Gemini API key used in this 
project have been removed and replaced with placeholders in the code. 
To run this project yourself, you'll need to generate your own keys:
- Telegram Bot Token: via [@BotFather](https://t.me/BotFather)
- Gemini API Key: via [Google AI Studio](https://aistudio.google.com/app/apikey)

Replace the placeholder values in `appp.py` with your own keys before running.
